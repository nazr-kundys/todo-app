A full-featured todo app built with **React** and **TypeScript**. Supports creating, editing, deleting, and filtering todos.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/nazr-kundys/todo-app.git
cd todo-app
```

### Install dependencies

```bash
npm install
```

### Run the project locally

```bash
npm start
```

## Features

- **CRUD operations** — add, edit, delete todos via a REST API with loading states
- **Bulk actions** — toggle all todos at once or clear all completed with one click
- **Filtering** — view All, Active, or Completed todos
- **Temp todo** — optimistic placeholder shown while a new todo is being saved
- **Error notifications** — auto-dismissing banners for all failure cases (load, add, update, delete)
- **Keyboard UX** — input auto-focuses on mount and after every action
